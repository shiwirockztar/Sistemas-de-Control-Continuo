\section{Introducción}

En el estudio de los sistemas de control es fundamental comprender el comportamiento dinámico de los procesos físicos antes de diseñar estrategias de control. Para ello se emplean técnicas de identificación de sistemas que permiten obtener modelos matemáticos a partir de datos experimentales. Estos modelos representan la relación entre las variables de entrada y salida del sistema y constituyen la base para el diseño y análisis de controladores.

En este laboratorio se utiliza el sistema experimental \textit{TempControlLab}, desarrollado para el estudio de procesos térmicos. Este dispositivo permite analizar la dinámica de transferencia de calor mediante el uso de calentadores eléctricos y sensores de temperatura. El sistema posee dos actuadores térmicos (calentadores) y dos sensores de temperatura, lo que lo convierte en un sistema de tipo MIMO (Multiple Input Multiple Output).

A través de la aplicación de señales de prueba tipo escalón y la medición de la respuesta térmica del sistema, se obtienen datos experimentales que permiten identificar modelos dinámicos del proceso. Posteriormente, estos modelos se ajustan mediante diferentes métodos de identificación, tales como Ziegler–Nichols, Smith y técnicas de optimización, con el fin de obtener una representación matemática adecuada del sistema.


\section{Propósito del laboratorio}

El objetivo principal de este laboratorio es realizar la identificación experimental de un proceso térmico utilizando el sistema TempControlLab. Para ello se aplican pruebas de escalón en los actuadores térmicos y se registran las respuestas de temperatura del sistema.

A partir de los datos experimentales obtenidos, se busca determinar modelos dinámicos que describan la relación entre las entradas del sistema (potencia de los calentadores) y sus salidas (temperaturas medidas por los sensores). En particular, se pretende ajustar modelos de tipo \textit{First Order Plus Dead Time} (FOPDT), los cuales son ampliamente utilizados en la modelación de procesos industriales.

De manera específica, los objetivos del laboratorio son:

\begin{itemize}
\item Analizar la dinámica térmica del sistema experimental TempControlLab.
\item Obtener datos experimentales mediante pruebas de escalón en los calentadores.
\item Identificar modelos dinámicos del proceso utilizando distintos métodos de identificación.
\item Comparar los resultados obtenidos mediante los métodos de Ziegler–Nichols, Smith y optimización.
\item Analizar la interacción entre las variables del sistema, evidenciando el comportamiento MIMO del proceso térmico.
\end{itemize}


\section{Modelo físico del sistema}

El sistema TempControlLab se basa en la transferencia de calor generada por resistencias eléctricas controladas mediante señales PWM provenientes de una tarjeta Arduino. Cada calentador produce energía térmica proporcional a la potencia eléctrica suministrada, lo que provoca un aumento en la temperatura de la placa metálica donde se encuentra instalado.

La dinámica térmica del sistema puede describirse mediante un balance de energía que considera los diferentes mecanismos de transferencia de calor presentes en el proceso. En general, el comportamiento térmico de cada elemento puede representarse mediante una ecuación diferencial del tipo

\begin{equation}
m c_p \frac{dT}{dt} = Q_{in} - Q_{out}
\end{equation}

donde $m$ corresponde a la masa del elemento térmico, $c_p$ es el calor específico del material y $T$ es la temperatura del sistema.

En el caso particular del TempControlLab, los principales mecanismos de transferencia de calor son:

\begin{itemize}
\item \textbf{Convección:} intercambio de calor entre el sistema y el ambiente circundante.
\item \textbf{Radiación térmica:} transferencia de energía entre superficies a diferentes temperaturas.
\item \textbf{Conducción:} transferencia de calor entre las dos placas térmicas del sistema.
\end{itemize}

El balance de energía para cada calentador puede representarse de forma simplificada como

\begin{equation}
m c_p \frac{dT_H}{dt} =
UA(T_{\infty}-T_H)
+ \epsilon \sigma A (T_{\infty}^4 - T_H^4)
+ Q_{C12}
+ Q_{R12}
+ \alpha Q
\end{equation}

donde $UA$ representa la transferencia por convección con el ambiente, $\epsilon \sigma A$ describe el efecto de radiación térmica, $Q_{C12}$ corresponde al flujo de calor por conducción entre los calentadores y $Q_{R12}$ representa el intercambio de calor por radiación entre las superficies.

Adicionalmente, los sensores de temperatura presentan una dinámica propia asociada a su inercia térmica, la cual puede modelarse como

\begin{equation}
\tau_s \frac{dT_C}{dt} = T_H - T_C
\end{equation}

donde $T_C$ corresponde a la temperatura medida por el sensor y $T_H$ es la temperatura real de la superficie calentada.

Debido a la interacción térmica entre los dos calentadores, el sistema presenta acoplamiento entre sus variables, lo que implica que una variación en uno de los calentadores puede afectar ambas temperaturas medidas. Por esta razón el sistema se modela como un sistema MIMO con dos entradas y dos salidas.


\section{Interpretación del modelo identificado}

A partir de los datos experimentales obtenidos mediante las pruebas de escalón, se identificaron modelos dinámicos del tipo FOPDT para cada una de las relaciones entrada-salida del sistema. En general, este tipo de modelo puede representarse como

\begin{equation}
G(s) = \frac{K}{\tau s + 1} e^{-\theta s}
\end{equation}

donde $K$ corresponde a la ganancia del proceso, $\tau$ es la constante de tiempo del sistema y $\theta$ representa el tiempo muerto asociado al proceso.

\section*{Análisis del Modelo FOPDT}

El modelo (FOPDT) es utilizado para describir sistemas que tienen un comportamiento de primer orden con retardo. En este análisis, aplicamos los siguientes pasos para lograr la identificación del modelo y estimar los parámetros de ganancia \( K \), tiempo muerto \( \theta \), y constante de tiempo \( \tau \).

\subsection*{Pasos para lograr la identificación del modelo FOPDT}
\begin{enumerate}
    \item \textbf{Identificar el escalón de entrada:} Analizamos los datos de entrada y determinamos el punto en el que la señal cambia, identificando el inicio del escalón.
    
    \item \textbf{Ajustar el modelo FOPDT:} Usamos un ajuste de regresión polinómica para modelar el comportamiento del sistema.
    
    \item \textbf{Aplicar los métodos de Ziegler-Nichols y Smith:} Estimamos los parámetros del modelo utilizando la respuesta al escalón y los métodos de identificación, obteniendo las variables \( K \), \( \theta \) y \( \tau \).
    
    \item \textbf{Simular la respuesta al escalón con el modelo FOPDT:} Utilizamos la función de transferencia para simular cómo responde el sistema considerando el tiempo muerto.
    
    \item \textbf{Comparar los resultados:} Comparamos la respuesta real, la regresión polinómica y la respuesta simulada con el modelo FOPDT.
\end{enumerate}

\subsection*{Resultados}

Se realizo un codigo en un Notebook de Jupyter en el cual se calcularon los valores de los modelos a partir de una simulación experimental de la planta termica con los codigos proporcionados con tclab. A continuación, se muestran los resultados obtenidos para los parámetros del modelo FOPDT tanto de Smith y Ziegler-Nichols para el calentador Q1:

\section*{Conclusiones}

A partir de la comparación de los modelos FOPDT obtenidos mediante los métodos de Ziegler-Nichols y Smith, podemos observar lo siguiente:

\begin{itemize}
    \item Los modelos ajustados a partir de los métodos de Ziegler-Nichols y Smith presentan un buen ajuste a los datos experimentales, con una diferencia en los valores de la constante de tiempo (\(\tau\)) y el tiempo muerto (\(\theta\)) entre ambos métodos.
    \item El modelo de Ziegler-Nichols muestra un tiempo muerto mayor (\(\theta = 2.95 \, \text{segundos}\)) y una constante de tiempo más grande (\(\tau = 159.05 \, \text{segundos}\)) en comparación con el modelo de Smith, que tiene un tiempo muerto de \( \theta = 11.59 \, \text{segundos} \) y una constante de tiempo \( \tau = 109.32 \, \text{segundos} \).
    \item Ambos modelos siguen de cerca la respuesta de los datos experimentales, pero el modelo de Smith (curva morada) parece ajustarse mejor a las muestras originales (puntos grises) a lo largo de toda la curva. 
    \item La diferencia en los tiempos muertos y las constantes de tiempo refleja cómo cada método ajusta la respuesta del sistema a la entrada escalón, lo que es crucial para la elección del modelo adecuado dependiendo de las necesidades del control térmico en el proceso.
    \item En general, ambos modelos son útiles para la simulación y el diseño de controladores, pero el modelo de Smith presenta un mejor ajuste a los datos originales, especialmente en los primeros y últimos tramos de la respuesta.
\end{itemize}

Se puede concluir que los métodos de identificación utilizados son adecuados para representar el comportamiento del sistema térmico estudiado. Sin embargo, el modelo de Smith ofrece una representación más precisa de la respuesta térmica en comparación con el modelo de Ziegler-Nichols, especialmente en la comparación visual con las muestras originales.
