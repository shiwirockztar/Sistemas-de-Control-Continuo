# Derivacion de Ecuaciones de Movimiento (Euler-Lagrange)

## Descripcion General

En este documento se deriva el modelo dinamico de dos coordenadas generalizadas $\theta_1$ y $\theta_2$ usando el metodo de Euler-Lagrange.

---

## 1. Lagrangiano del Sistema

Se parte del lagrangiano:

$$
L = \frac{1}{2} m_1 l^2 \dot{\theta}_1^2
  + \frac{1}{2} m_2 l^2 \dot{\theta}_2^2
  - gl\left[m_1(1-\cos\theta_1) + m_2(1-\cos\theta_2)\right]
  - \frac{1}{2}ka^2(\sin\theta_2 - \sin\theta_1)^2
$$

```text
L = (1/2) m1 l^2 (theta1_punto)^2
  + (1/2) m2 l^2 (theta2_punto)^2
  - g l [ m1 (1 - cos(theta1)) + m2 (1 - cos(theta2)) ]
  - (1/2) k a^2 ( sin(theta2) - sin(theta1) )^2
```

La ecuacion de Euler-Lagrange para cada coordenada $\theta_i$ es:

$$
\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{\theta}_i}\right)
- \frac{\partial L}{\partial \theta_i} = 0,
\quad i = 1,2
$$

```text
d/dt ( dL/d(theta_i_punto) ) - dL/d(theta_i) = 0,   i = 1,2
```

---

## 2. Desarrollo para $\theta_1$

### Derivada respecto a $\dot{\theta}_1$

$$
\frac{\partial L}{\partial \dot{\theta}_1} = m_1 l^2\dot{\theta}_1
$$

```text
dL/d(theta1_punto) = m1 l^2 theta1_punto
```

$$
\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{\theta}_1}\right) = m_1 l^2\ddot{\theta}_1
$$

```text
d/dt [ dL/d(theta1_punto) ] = m1 l^2 theta1_doble_punto
```

### Derivada respecto a $\theta_1$

$$
\frac{\partial L}{\partial \theta_1}
= -m_1gl\sin\theta_1
+ ka^2(\sin\theta_2 - \sin\theta_1)\cos\theta_1
$$

```text
dL/d(theta1) = -m1 g l sin(theta1)
              + k a^2 ( sin(theta2) - sin(theta1) ) cos(theta1)
```

### Ecuacion resultante para $\theta_1$

$$
m_1l^2\ddot{\theta}_1
+ m_1gl\sin\theta_1
- ka^2(\sin\theta_2 - \sin\theta_1)\cos\theta_1 = 0
$$

```text
m1 l^2 theta1_doble_punto
+ m1 g l sin(theta1)
- k a^2 ( sin(theta2) - sin(theta1) ) cos(theta1) = 0
```

---

## 3. Desarrollo para $\theta_2$

### Derivada respecto a $\dot{\theta}_2$

$$
\frac{\partial L}{\partial \dot{\theta}_2} = m_2 l^2\dot{\theta}_2
$$

```text
dL/d(theta2_punto) = m2 l^2 theta2_punto
```

$$
\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{\theta}_2}\right) = m_2 l^2\ddot{\theta}_2
$$

```text
d/dt [ dL/d(theta2_punto) ] = m2 l^2 theta2_doble_punto
```

### Derivada respecto a $\theta_2$

$$
\frac{\partial L}{\partial \theta_2}
= -m_2gl\sin\theta_2
- ka^2(\sin\theta_2 - \sin\theta_1)\cos\theta_2
$$

```text
dL/d(theta2) = -m2 g l sin(theta2)
              - k a^2 ( sin(theta2) - sin(theta1) ) cos(theta2)
```

### Ecuacion resultante para $\theta_2$

$$
m_2l^2\ddot{\theta}_2
+ m_2gl\sin\theta_2
+ ka^2(\sin\theta_2 - \sin\theta_1)\cos\theta_2 = 0
$$

```text
m2 l^2 theta2_doble_punto
+ m2 g l sin(theta2)
+ k a^2 ( sin(theta2) - sin(theta1) ) cos(theta2) = 0
```

---

## 4. Sistema Final No Lineal

$$
\begin{cases}
m_1l^2\ddot{\theta}_1 + m_1gl\sin\theta_1 - ka^2(\sin\theta_2 - \sin\theta_1)\cos\theta_1 = 0 \\
m_2l^2\ddot{\theta}_2 + m_2gl\sin\theta_2 + ka^2(\sin\theta_2 - \sin\theta_1)\cos\theta_2 = 0
\end{cases}
$$

```text
m1 l^2 theta1_doble_punto + m1 g l sin(theta1)
- k a^2 ( sin(theta2) - sin(theta1) ) cos(theta1) = 0

m2 l^2 theta2_doble_punto + m2 g l sin(theta2)
+ k a^2 ( sin(theta2) - sin(theta1) ) cos(theta2) = 0
```

---

## 5. Aproximacion de Angulo Pequeno (Opcional)

Si $\sin\theta \approx \theta$ y $\cos\theta \approx 1$, el sistema linealizado queda:

$$
\begin{cases}
m_1l^2\ddot{\theta}_1 + (m_1gl + ka^2)\theta_1 - ka^2\theta_2 = 0 \\
m_2l^2\ddot{\theta}_2 + (m_2gl + ka^2)\theta_2 - ka^2\theta_1 = 0
\end{cases}
$$

```text
m1 l^2 theta1_doble_punto + (m1 g l + k a^2) theta1 - k a^2 theta2 = 0
m2 l^2 theta2_doble_punto + (m2 g l + k a^2) theta2 - k a^2 theta1 = 0
```
