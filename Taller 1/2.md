# Derivacion de ecuaciones de movimiento (Euler-Lagrange)

## 1) Lagrangiano del sistema

Se toma el lagrangiano:

```text
L = (1/2) m1 l^2 (theta1_punto)^2
  + (1/2) m2 l^2 (theta2_punto)^2
  - g l [ m1 (1 - cos(theta1)) + m2 (1 - cos(theta2)) ]
  - (1/2) k a^2 ( sin(theta2) - sin(theta1) )^2
```

Las ecuaciones de Euler-Lagrange para cada coordenada generalizada theta_i son:

```text
d/dt ( dL/d(theta_i_punto) ) - dL/d(theta_i) = 0,   i = 1, 2
```

## 2) Ecuacion para theta1

Derivada respecto a la velocidad:

```text
dL/d(theta1_punto) = m1 l^2 theta1_punto
d/dt[dL/d(theta1_punto)] = m1 l^2 theta1_doble_punto
```

Derivada respecto a la coordenada:

```text
dL/d(theta1) = -m1 g l sin(theta1)
              + k a^2 ( sin(theta2) - sin(theta1) ) cos(theta1)
```

Sustituyendo en Euler-Lagrange:

```text
m1 l^2 theta1_doble_punto
+ m1 g l sin(theta1)
- k a^2 ( sin(theta2) - sin(theta1) ) cos(theta1) = 0
```

## 3) Ecuacion para theta2

Derivada respecto a la velocidad:

```text
dL/d(theta2_punto) = m2 l^2 theta2_punto
d/dt[dL/d(theta2_punto)] = m2 l^2 theta2_doble_punto
```

Derivada respecto a la coordenada:

```text
dL/d(theta2) = -m2 g l sin(theta2)
              - k a^2 ( sin(theta2) - sin(theta1) ) cos(theta2)
```

Sustituyendo en Euler-Lagrange:

```text
m2 l^2 theta2_doble_punto
+ m2 g l sin(theta2)
+ k a^2 ( sin(theta2) - sin(theta1) ) cos(theta2) = 0
```

## 4) Forma final no lineal

```text
m1 l^2 theta1_doble_punto + m1 g l sin(theta1)
- k a^2 ( sin(theta2) - sin(theta1) ) cos(theta1) = 0

m2 l^2 theta2_doble_punto + m2 g l sin(theta2)
+ k a^2 ( sin(theta2) - sin(theta1) ) cos(theta2) = 0
```

## 5) Aproximacion de angulo pequeno (opcional)

Usando sin(theta) ~ theta y cos(theta) ~ 1:

```text
m1 l^2 theta1_doble_punto + (m1 g l + k a^2) theta1 - k a^2 theta2 = 0
m2 l^2 theta2_doble_punto + (m2 g l + k a^2) theta2 - k a^2 theta1 = 0
```
