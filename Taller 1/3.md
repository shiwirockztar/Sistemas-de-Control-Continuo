# 📘 Sistema de Péndulos Acoplados

Este proyecto modela un sistema de **dos péndulos acoplados mediante un resorte**, utilizando el método de Lagrange para obtener sus ecuaciones de movimiento.

---

## ⚙️ Lagrangiano del sistema

El Lagrangiano está dado por:


L = (1/2)m1l^2theta1_dot^2 + (1/2)m2l^2theta2_dot^2
- gl[ m1*(1 - cos(theta1)) + m2*(1 - cos(theta2)) ]
- (1/2)ka^2*(sin(theta2) - sin(theta1))^2


---

## 🧮 Ecuaciones de Euler-Lagrange

Para cada coordenada generalizada:


d/dt(∂L/∂theta_dot) - ∂L/∂theta = 0


---

# 🔹 Derivación para θ1

### 1. Derivada respecto a theta1_dot


∂L/∂theta1_dot = m1l^2theta1_dot

d/dt = m1l^2theta1_ddot


---

### 2. Derivada respecto a theta1

#### Parte gravitacional:

m1gl*sin(theta1)

#### Parte del resorte:


ka^2(sin(theta2) - sin(theta1))*cos(theta1)


---

### 3. Sustitución


m1l^2theta1_ddot + m1gl*sin(theta1)

ka^2(sin(theta2) - sin(theta1))*cos(theta1) = 0

---

# 🔹 Derivación para θ2

### 1. Derivada respecto a theta2_dot


∂L/∂theta2_dot = m2l^2theta2_dot

d/dt = m2l^2theta2_ddot


---

### 2. Derivada respecto a theta2

#### Parte gravitacional:

m2gl*sin(theta2)

#### Parte del resorte:

ka^2(sin(theta2) - sin(theta1))*cos(theta2)

---

### 3. Sustitución


m2l^2theta2_ddot + m2gl*sin(theta2)

ka^2(sin(theta2) - sin(theta1))*cos(theta2) = 0

---

# 🔗 Sistema final


m1l^2theta1_ddot + m1gl*sin(theta1)

ka^2(sin(theta2) - sin(theta1))*cos(theta1) = 0

m2l^2theta2_ddot + m2gl*sin(theta2)

ka^2(sin(theta2) - sin(theta1))*cos(theta2) = 0

---

## 💡 Interpretación

- Cada péndulo tiene dinámica propia debido a la gravedad.
- El resorte introduce **acoplamiento no lineal**.
- El sistema es **no lineal** por los términos `sin` y `cos`.
- Dependiendo de parámetros, pueden aparecer fenómenos como:
  - sincronización
  - oscilaciones acopladas
  - comportamiento complejo
